# User Guide Screenshots

Screenshots referenced by `docs/user_guide.md`. Each screenshot should be captured at
1280x800 resolution or similar, showing the feature in context.

## Required Screenshots

| Filename | Section | What to Capture |
|----------|---------|-----------------|
| `asset-browser.png` | Asset Browser | Explore page with sidebar showing expanded database > table > columns hierarchy |
| `lineage-graph.png` | Lineage Graph | A column-level lineage graph showing multiple table nodes with edges and highlighted path |
| `lineage-graph-highlighted.png` | Path Highlighting | Same graph with a column selected showing highlighted path and dimmed non-path nodes |
| `toolbar-controls.png` | Toolbar Controls | The toolbar showing direction dropdown, depth slider, fit-to-selection, export, and view mode toggle |
| `detail-panel-columns.png` | Detail Panel | Detail panel open on Columns tab showing column list with types and lineage counts |
| `detail-panel-statistics.png` | Detail Panel | Detail panel open on Statistics tab showing table metadata |
| `detail-panel-ddl.png` | Detail Panel | Detail panel open on DDL tab showing syntax-highlighted SQL |
| `loading-progress.png` | Loading Progress | The loading progress bar showing a stage message and timing information |
| `search-results.png` | Search | Search results page showing grouped results with expanded items |
| `database-lineage.png` | Database Lineage | Database-level lineage view with Load More button and table count indicator |

## Capture Tips

- Use a browser at 1280x800 or similar resolution
- Use the demo/test data so screenshots are reproducible
- Ensure dark mode is off (default light theme)
- Crop browser chrome if possible (show only the app UI)
